# Java Assignment 2 – Lunch Order Tracker

## Description
This project is a Java-based application that simulates a lunch order tracking system.  
It allows multiple lunch orders to be created, stored in an array, and displayed using object-oriented programming concepts.

## Features
- Create multiple Lunchbox objects
- Store objects in an array
- Display order details using methods
- Track delivery status (Delivered / Not Delivered)
- Use of constructors, getters/setters, and methods

## Technologies Used
- Java
- VS Code
- Git & GitHub

## How It Works
The program creates multiple lunch orders and stores them in an array.  
Each order can be marked as delivered and displayed using a loop.
